# StudentDB SQL Project

## Overview
This is a basic SQL project demonstrating CRUD operations in Microsoft SQL Server (SSMS).  
The project includes:

- Creating a database (`StudentDB`)
- Creating a table (`Students`)
- Inserting sample data
- Selecting data
- Updating records
- Deleting records

## Technologies
- SQL Server Management Studio (SSMS)
- T-SQL

## How to Run
1. Open SSMS.
2. Open the `StudentDB_Project.sql` file.
3. Execute all queries step by step.
4. Check results using `SELECT * FROM Students;`.
